## What It Solves

HiPay has a 12-month window to capture merchants fleeing Worldline's compliance scandal.
IdomePay is the technical infrastructure that makes that migration frictionless, fast, and auditable.

## The 5 Problems It Solves

| Problem | IdomePay Solution |
|---|---|
| Token lock-in (merchants can't leave Worldline) | PSP-agnostic token vault simulation |
| Cold start (Sentinel has no data on new merchants) | Transfer learning from similar merchant profiles |
| Black box fraud decisions (EU AI Act violation) | SHAP explainability on every Sentinel decision |
| False positive rate 10–15% (lost revenue) | Sentinel pre-calibrated → 3.1% with flywheel |
| No proof of compliance (post-Worldline scandal) | 100% audit log, auto-generated, SHA-256 hashed |

## The 5 Tabs

| Tab | What It Shows |
